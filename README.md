# Pokemon "Malware" Pet

A Godot Project based around the idea of desktop pets with the implementation of pokemon

* Controllable pokemon trainer
* Gender selection
* Wandering NPC
* NPC might take something on the desktop I dont know
* Has a file storage system for some reason
* Pokemon Battle simulator
